# Changelog

## 0.1.0

- Initial release: Token theme (`TuTheme` / `TuColors` / `TuTextStyles`) with built-in defaults and host override.
- Primitives: Button, Tag, TextField, IconButton, BadgeWrapper, NotificationDot, Divider.
- Feedback: BottomSheet, ActionSheet, ActionMenu, Empty, Error, ConfirmDialog, BlockingLoadingOverlay.
- Layout: AppBar, BackButton, ListItem, PaginatedList, RefreshControl.
- Motion: SwipeCardStack, SwipeActionButton, CarouselIndicator.
- Decoration: DashedBorder.
- Example Gallery + brand token demo.
